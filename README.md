## Xcode

### `#include<bits/stdc++.h>` 사용하기.

- m1, m4 기준

```shell
cd /Applications/Xcode.app/Contents/Developer/Platforms/MacOSX.platform/Developer/SDKs/MacOSX.sdk/usr/include/c++/v1
mkdir bits
cd bits
vim stdc++.h
```

[header/stdc++.h](header/stdc++.h) file 을 붙여넣는다.
